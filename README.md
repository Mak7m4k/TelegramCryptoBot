# TelegramCryptoBot
  
## Описание:  
Telegram бот использующий API CoinMarcetCap.  
Умеет:  
  Выводить информацию о криптовалюте.  
  Выводить курс монет.  
  
##Библиотеки:
```
telebot
requests
```

## Запуск:  
  Найти строку ```TKEN = ''``` вписать туда ключ TG бота.  
  Найти ```'X-CMC_PRO_API_KEY': ''``` вписать туда ключ CoinMarketCap.
